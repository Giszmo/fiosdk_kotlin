FIOSDK(privateKey: String, publicKey: String, technologyPartnerId: String, serializationProvider: ISerializationProvider, signatureProvider: ISignatureProvider, networkBaseUrl: String)

Kotlin SDK for FIO Foundation API

Parameters

privateKey - the fio private key of the client sending requests to FIO API.

publicKey - the fio public key of the client sending requests to FIO API.

technologyPartnerId - FIO Address of the wallet which generates this transaction. Set to empty if not known.

serializationProvider - the serialization provider used for abi serialization and deserialization.

signatureProvider - the signature provider used to sign block chain transactions.

networkBaseUrl - the url to the FIO API.
